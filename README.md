# English Teachers Planner - PDF Only Version

Planificador de clases de inglés para MEDUCA Panamá. Exportación a PDF 100% cliente.

## Características

✅ Exportación a PDF sin backend  
✅ Funciona offline (PWA)  
✅ Formato oficial MEDUCA  
✅ Edición en tiempo real  
✅ Soporte bilingüe (ES/EN)  

## Instalación

```bash
cd frontend
npm install
npm run dev
```

## Deploy en Vercel

1. Conectar repositorio
2. Root Directory: `frontend`
3. Build Command: `npm run build`
4. Output Directory: `dist`
